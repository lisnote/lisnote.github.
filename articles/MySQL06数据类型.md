# 06数据类型

| 类型      | 大小或格式             | 作用         |
| --------- | ---------------------- | ------------ |
| tinyint   | 1字节                  | 十分小的整数 |
| smallint  | 2字节                  | 较小的整数   |
| int       | 4字节                  | 标准整数     |
| bigint    | 8字节                  | 大整数       |
| float     | 4字节                  | 单精度浮点数 |
| double    | 8字节                  | 双精度浮点数 |
| decimal   | x                      | 精确浮点数   |
| char      | 0-255                  | 定长字符串   |
| varchar   | 0-65535                | 不定长字符串 |
| tinytext  | 2^8-1                  | 微型文本     |
| text      | 2^16-1                 | 文本串       |
| date      | YYYY-MM-DD             | 日期格式     |
| time      | HH:mm:ss               | 时间格式     |
| datetime  | YYYY-MM-DD HH:mm:ss    | 日期时间格式 |
| timestamp | 1970.1.1到现在的毫秒数 | 时间戳       |

* decimal(M,D)
	M -> 指定小数点左边和右边可以存储的十进制数字的最大个数，   （M的范围是0~38）
	D -> 小数点右侧，小数位的长度，不能超过D，超过部分四舍五入，（D的范围是0~30）
