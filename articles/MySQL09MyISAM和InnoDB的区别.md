# 09MyISAM和InnoDB的区别

|                  | MYISAM | INNODB         |
| ---------------- | ------ | -------------- |
| 事务支持         | ×      | √              |
| 数据行锁定       | 表锁定 | √              |
| 外键约束         | ×      | √              |
| 全文索引         | ×      | √              |
| 表空间大小       | 较小   | 约MYISAM两倍   |
| 自增量的存储位置 | 文件   | 内存(断电归零) |

* 常规操作
  * MYISAM : 节约空间,速度更快
  * INNODB : 安全性高,事务处理,多表多用户操作
