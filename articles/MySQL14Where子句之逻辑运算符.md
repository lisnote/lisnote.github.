# 14Where子句之逻辑运算符

* 作用 : 检索数据中符合条件的值

| 符号     | 说明     |
| -------- | -------- |
| =        | 等于     |
| <> 或 != | 不等于   |
| >        | 大于     |
| >=       | 大于等于 |
| <        | 小于     |
| <=       | 小于等于 |
| AND      | 与       |
| OR       | 或       |
| NOT      | 非       |

```
SELECT score
FROM score
WHERE score >= 80;
```
